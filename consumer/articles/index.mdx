# Alexandria

Files are located at /Users/chandlerprall/projects/lilith-projects/alexandria

## About

This is the project for Chandler's personal website. It is built with a custom static site generator, called Alexandria. You'll notice `alexandria` is the project directory - this project contains both the generator (alexandria directory) and the site (consumer directory). We use yarn to manage and link the two workspaces.


    
## Build and Run Steps

Compiling alexandria
* `cd ./alexandria`
* `yarn compile`

Compiling consumer
* `cd ./consumer`
* `yarn tsc && node .`

Serving
* `cd ./consumer/out/build`
* `python -m http.server`

# Alexandria Project Structure

## alexandria Directory
- **package.json**: Contains dependencies and scripts for the generator.
- **tsconfig.json**: TypeScript configuration for the generator.
- **src/**: Source code for the generator.

## consumer Directory
- **package.json**: Contains dependencies and scripts for the consumer.
- **template.html**: HTML template for the site.
- **tsconfig.json**: TypeScript configuration for the consumer.
- **src/**: Source code for the consumer.
- **public/**: Static assets for the site.
- **layouts/**: Layout components for the site.
- **components/**: Reusable components for the site.
- **articles/**: Markdown articles for the site.

## Key Files
- **alexandria/package.json**: Lists dependencies and scripts for the generator.
- **alexandria/tsconfig.json**: Configures TypeScript for the generator.
- **consumer/package.json**: Lists dependencies and scripts for the consumer.
- **consumer/template.html**: HTML template for the site.
- **consumer/tsconfig.json**: Configures TypeScript for the consumer.
- **consumer/src/index.mjs**: Main entry point for the consumer.
- **consumer/public/site.css**: Static CSS for the site.
- **consumer/layouts/main.mdx**: Main layout for the site.
- **consumer/components/Header.tsx**: Header component.
- **consumer/components/PageCentered.tsx**: Centered page component.
- **consumer/components/_header.tsx**: Header component.
- **consumer/articles/about.mdx**: About article.
- **consumer/articles/index.mdx**: Index article.
