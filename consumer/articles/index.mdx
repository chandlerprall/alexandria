# Welcome to Alexandria

Alexandria is a custom static site generator built for Chandler's personal website. This project is divided into two main directories: `alexandria` and `consumer`.

## Alexandria Directory

The `alexandria` directory contains the source code for the build tool. It includes:
- `build`: Contains build-related files.
- `node_modules`: Contains the project dependencies.
- `package.json`: Contains the project metadata and dependencies.
- `src`: Contains the source code for the build tool.
- `tsconfig.json`: Contains the TypeScript configuration.

## Consumer Directory

The `consumer` directory contains the source code for the site definition. It includes:
- `articles`: Contains articles written in MDX.
- `build`: Contains build-related files.
- `components`: Contains custom components.
- `layouts`: Contains layout components.
- `out`: Contains the output of the build process.
- `package.json`: Contains the project metadata and dependencies.
- `src`: Contains the source code for the site.
- `template.html`: Contains the HTML template for the site.
- `tsconfig.json`: Contains the TypeScript configuration.

## Getting Started

To get started with the Alexandria project, follow these steps:
1. Clone the repository.
2. Install dependencies using `yarn`.
3. Build the site using `yarn build`.
4. Serve the site using `yarn serve`.

## Features

- **Custom Static Site Generator**: Built with TypeScript for robustness and maintainability.
- **MDX Support**: Write articles and pages using MDX, a JSX-flavored Markdown.
- **Custom Components**: Reusable components for consistent design and functionality.
- **Responsive Design**: Ensures the site looks great on all devices.
